# Universal Electronics TikTok Solar Stories (9:16 Vertical Content Suite)

## What is Included
- `index.html` — Interactive TikTok & Reels one-page web application featuring phone frame mockups, teleprompter mode, category filters, and an off-grid solar calculator.
- `styles.css` — Ultra-modern responsive styling with Google Fonts (`Plus Jakarta Sans` & `Outfit`), glassmorphism, 9:16 aspect ratios, and smooth animations.
- `script.js` — Category filtering, interactive solar battery sizing calculator, TikTok teleprompter script modal, and clipboard toast notifications.
- `CONTENT.md` — All seven long-form solar stories, 90-second video scripts, captions, and hashtags.
- `IMAGE_PROMPTS.md` — 21 production-grade Midjourney v6 (`--ar 9:16`) and DALL-E 3 prompts with safe-zone directives and color palettes.
- `assets/images/` — 7 photorealistic high-resolution 9:16 vertical AI generated key visuals (`01-becquerel-tiktok.jpg` to `07-off-grid-tiktok.jpg`).
- `assets/universal-logo.png` — Official Universal Electronics Ethiopia logo.

## Open the Website
Double-click `index.html`. No build step or server installation is required.

## Key Features
1. **TikTok 9:16 Phone Mockup Previews**: Displays video key visuals framed inside sleek smartphone frames.
2. **Reel Script Teleprompter**: Click "▶ Play Script" on any story to open a full-screen video narration rehearsal view.
3. **Interactive Solar Calculator**: Select household appliances to dynamically calculate continuous Wattage, daily kWh, lithium battery capacity, and required solar PV array size.
4. **Category Filter Tabs**: Filter stories instantly by topic (History, Space, Human Ingenuity, Solar Scale, Everyday Science, Thought Experiment, Off-Grid Challenge).

## Recommended Image & Video Export Sizes
- TikTok / Instagram Reels / YouTube Shorts: **1080 × 1920 (9:16 vertical)**
- Format: JPG / PNG / WebP

## Deployment
Static website—can be deployed immediately to Cloudflare Pages, GitHub Pages, Vercel, Netlify, or any standard web server.
